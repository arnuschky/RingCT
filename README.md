## Installation 
(Tested on Ubuntu 16)
I may have forgotten some dependencies. 
```
sudo apt install g++
sudo apt install cmake
sudo apt install libboost-dev
sudo apt install libboost-test-dev
sudo apt install libboost-date-time-dev
sudo apt install libboost-regex-dev
```

Run cmake and make
```
cmake ./
make
./rct
```
